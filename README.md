# Testes Automatizados


Requisitos: 

 Postman instalado.

  -  Orientações para visualizar os scripts de testes e localizar o arquivo JSON no Github:

1 - Baixar o arquivo:  

https://github.com/KellyRabelo/projetos/blob/master/Projeto%20PostmanTestes%20Automatizados.rar

2  - Abrir o postman;

3 - Importar o arquivo Json baixado no caminho da pasta:  \Projeto PostmanTestes Automatizados\Exportação Json Postman\ColeçãoPostman.postman_collection.json

4 - Visualizar e executar a coleção de requisições;

-  Orientações para visualizar Relatório Falha/Acertos e Evidências dos testes executados: 

1 - Relatório Falha/Acertos pode ser visualizado no recurso Monitor criado e configurado no Postman na “ColeçãoPostman” vindo do arquivo Json importado; 

2 - Evidências em imagens dos relatórios Falha/Acertos estão no arquivo baixado no caminho da pasta \Projeto PostmanTestes Automatizados\Relatório Monitoramento FalhasErros Postman;


3 - Evidências em imagens dos Runs dos scripts construidos no caminho da pasta  \Projeto PostmanTestes Automatizados\Prints dos Testes Executavel Automatizados Postman.
